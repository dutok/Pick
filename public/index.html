Support

  • 1. How do I update config files?

    You can visit the File Editor and find the file you wish to edit. Click the file and a dialog should pop-up with the contents of the file. Edit the textbox and then click "Update" when you're ready to re-upload it to the server.

  • 2. How do I view and send chat messages?

    Any player chat on the server is sent to the Chat tab. Any messages sent from here are broadcasted to the server.

  • 3. Why can't I see any 'say' commands in the console?

    All 'say' commands are automatically forwarded to the Chat tab for easier viewing.

  • 4. How do I logout?

    You can logout using the 'logout' link present in the top of the navigation bar or in the side navigation bar depending on the device you're browsing with.
